1. O que é o CORS?
    Tecnologia para determinar uma política de compatilhamento de recursos
    entre a máquina servidora e outras máquinas que desejam acessar seus
    recuros através de um navagador.


2. Qual finalidade de uma classe de configuração CORS para Spring Boot?
    Para definir de forma clara, utilizando métodos Java, as propriedades do CORS.

3. Qual a forma de configuração CORS utilizada nesta aula?
    Utilizamos uma classe de configuração, essa utiliza a annotation @Configuration,
    implementa a interface WebMvcConfigurer e o método addCorsMappings.
